# Bus Tracking System - Final Year Project (United Bus Tracking System (UBTS))

Real-time GPS-based tracking solution developed for UCER College Buses (Route 3 & 6) to assist students and staff in daily commuting.

## Project Structure
- **Driver_App:** Used by bus drivers to broadcast live GPS coordinates.
- **User_App:** Used by commuters to track bus location on Google Maps.
- **College_Admin:** Used by transport officers to monitor fleet status and breakdown reports.

## Features
- Real-time location tracking using GPS.
- Route monitoring for specific college bus routes.
- Simplified interface for immediate location updates.

## Tech Stack
- Android SDK
- Google Maps API
- GPS Services
